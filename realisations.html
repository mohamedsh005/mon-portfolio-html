Mes Compétences Clés

Synthèse des compétences techniques utilisées dans le cadre de ces projets :

  • **HTML/CSS :** Structuration de pages et design web.
  • **Développement Web :** Utilisation de WordPress et intégration/modification de code (PHP/JavaScript) pour la personnalisation.
  • **Versionnement :** Maîtrise de Git pour le suivi et le déploiement via GitHub Pages.
  • **Prototypage :** Conception d'applications mobiles (UI/UX) et de sites web.

Détail de mes Projets Personnels & Clients

1. Site Web Client : Heinsen-Prono.com

**Objectif :** Création d'un site professionnel pour un client dans le domaine du pronostic. Le projet a nécessité la mise en place d'une plateforme fonctionnelle et esthétique, ainsi que l'intégration de contenu dynamique.

**Technologies :** Mise en œuvre de **WordPress** pour la gestion de contenu, complétée par la modification de **code HTML/CSS/PHP** pour des ajustements de design spécifiques.

2. Application Mobile : Loopa (Suivi d'Habitudes) - *En développement*

**Objectif :** Conception et développement d'une application mobile dédiée au suivi et à la création d'habitudes. Ce projet est axé sur l'ergonomie (UX/UI) pour offrir une expérience utilisateur fluide et motivante.

**Statut :** Application en cours de développement, principalement axée sur le design d'interface et la logique de suivi.

3. Application : AI Tutor (Tuteur IA) - *En développement*

**Objectif :** Développer une application d'aide à l'étude utilisant des technologies d'intelligence artificielle (IA) pour fournir des explications personnalisées. Le défi est l'intégration d'une API d'IA et la structuration des réponses.

**Domaines explorés :** Développement d'interface utilisateur, intégration d'API externes, logique de l'application.

4. Ce Portfolio (Site web personnel)

**Objectif :** Démontrer ma capacité à structurer, styler et déployer un site web statique multi-pages de A à Z.

**Outils :** HTML, CSS, et déploiement via GitHub Pages.
