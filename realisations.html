Mes Compétences Clés

Synthèse des compétences techniques utilisées dans le cadre de ces projets :

  • **HTML/CSS :** Structuration de pages et design web.
  • **Développement Web :** Utilisation de WordPress et intégration/modification de code (PHP/JavaScript).
  • **Versionnement :** Maîtrise de Git pour le suivi et le déploiement via GitHub Pages.
  • **Prototypage :** Conception d'applications mobiles (UI/UX).

Détail de mes Projets Personnels & Clients

1. Site Web Client : Heinsen-Prono.com

**Objectif :** Création d'un site professionnel pour un client dans le domaine du pronostic. Le projet a nécessité la mise en place d'une plateforme fonctionnelle sur **WordPress**, avec personnalisation en **HTML/CSS/PHP**.

2. Application Mobile : Loopa (Suivi d'Habitudes) - *En développement*

**Fonction :** **Loopa est une application de suivi d'habitudes** (Habit Tracker). L'objectif est de motiver l'utilisateur à créer et maintenir de bonnes habitudes grâce à une interface intuitive.

**Statut :** Application en cours de développement, axée sur le design d'interface et la logique de suivi.

3. Application : AI Tutor (Tuteur IA) - *En développement*

**Fonction :** **AI Tutor est une application d'aide à l'étude basée sur l'IA**. Elle permet de **résumer des textes complexes** et d'analyser le contenu d'images (photos prises ou téléchargées) pour fournir des explications ciblées.

**Domaines explorés :** Développement d'interface utilisateur, gestion des requêtes API externes pour l'analyse de texte et d'image, et logique d'interaction avec le téléphone.

4. Ce Portfolio (Site web personnel)

**Objectif :** Démontrer ma capacité à structurer, styler et déployer un site web statique multi-pages de A à Z.

**Outils :** HTML, CSS, et déploiement via GitHub Pages.
